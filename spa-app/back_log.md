# Backlog / Идеи для разработки

1. **Рефакторинг**: Оптимизация структуры компонентов и типизации.
2. **Внедрение ИИ агента**: Интеграция умного помощника для путешествия.
3. **Развитие игровой логики**: Новые механики начисления баллов и достижений.
4. **Финальный клип**: Создание итогового видео из материалов поездки.
5. **Логика открытия картинки квиза**: Доработка механики разблокировки визуального контента при ответах.
